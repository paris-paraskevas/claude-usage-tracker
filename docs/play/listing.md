# Play Store listing copy

**App name** (≤30): `Claude Usage Tracker`

**Short description** (≤80):
`Watch your Claude Code 5h & weekly limits, sessions and context on your phone.`

**Full description** (≤4000):
```
Claude Usage Tracker shows your Claude Code usage on your phone — the same 5-hour and
weekly limits the /usage command shows — plus your active sessions, context window, and
all-time stats, with optional push alerts when you're near a limit.

It pairs with the free desktop app (Windows): open Settings → Remote, and scan the QR.
Your phone then mirrors your live usage from anywhere.

FEATURES
• 5-hour and weekly usage with live reset countdowns
• A one-glance verdict (All clear / Ease up / Near limit)
• Per-session context-window fill and token usage
• All-time totals: tokens, sessions, messages, streaks, peak hour, favorite model
• Anthropic service status (Ok / Errors / Down)
• Optional push notifications for usage thresholds, burn-rate, and context alerts

PRIVATE BY DESIGN
• End-to-end encrypted: the relay only ever stores ciphertext — nobody but your own
  devices can read your data.
• Your Claude login never leaves your computer.
• No ads, no analytics, no trackers.

REQUIREMENTS
• The free Claude Usage Tracker desktop app (the source of the data).
• Works while your desktop is running and online.

Open source (MIT): github.com/paris-paraskevas/claude-usage-tracker
```
